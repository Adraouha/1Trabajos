## pop-up anuncio

**Instructions** 
1- Crear una VENTANA EMERGENTE con un anuncio que aparezca cuando:

    opción 1: se clicke un botón.

    opción 2: cuando pasen 5 segundos.

La ventana está escondida (display
) hasta que se active el botón. o pasen 5 segundos. Puedes clickar sobre ella para para cerrarla.

Usar html-css-javascript. El css arranca en display
, y el javascript activa y desactiva el display (block para ver, u otros como flex).

2- MOVER CAJAS

Estás en un almacén y tienes que mover tres cajas con una palanca.

Cuando haces click al botón-palanca, se mueven las tres cajas.

Puedes parar la animación de diferentes maneras:

    Al apretar otro botón o sobre alguna de las cajas.

    Al apretar la misma palanca (usar TRUE/FALSE para esto)

    Al cabo de 4 segundos moviéndose, se paran solas (setTimeout)

+EXTRA:

3- SEMAFORO!

Tienes tres circulos en vertical y un color que va pasando de uno a otro. Cuando llegue al tercero, luego pasa al primero. Cómo lo resuelves? (no es tan fácil como parece)



# DIA 3 - ACCESIBILIDAD:
## Ejercicio:

**Crear desde cero una página que contenga:**

1- Menú de navegación: que haya una opción oculta que nos permita SALTAR el menú
Que haya otra opción visible que nos permita ir a otro lugar de la página

2- Al menos dos imágenes. Describir el contenido de la imagen con ALT

3- Párrafos de texto antes y después de las imágenes. Pueden ser Lorem Ipsum
Tener en cuenta las jerarquías y orden de la página HTML.

Usar CSS para esconder vínculos que deben ser ocultos.
Comprobar que se lee correctamente y da las opciones necesarias al activar el
NARRADOR del sistema.
